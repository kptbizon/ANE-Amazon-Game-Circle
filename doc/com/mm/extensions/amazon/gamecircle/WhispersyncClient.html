Classpublic final class WhispersyncClient
InheritanceWhispersyncClient Object

The Amazon Games WhispersyncClient Service client object. Whispersync data is always accessible, even when the player is not registered with Amazon or is offline. Unlike the LeaderboardsClient and AchievementsClient, WhispersyncClient can be used immediately after calling AmazonGamesClient.getInstance().initialize().

flush()method
public function flush():void

Forces game data to be written to the device’s file system. You don’t need to call flush() each time you change game data, because changes are automatically persisted to local storage via a background thread. However, if you know your app is about to close, calling flush() will save your game data by performing a blocking write to the file system.
