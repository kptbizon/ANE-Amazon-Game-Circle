getAchievements()method 
public function getAchievements():void

Returns a list of Achievement objects to the IGetAchievementsCallback callback. The list returned in the response includes all visible achievements for the game. Visible achievements include those that were never hidden, and those that were once hidden but since made visible. Each Achievement object in the list includes the current players progress toward the Achievement.

getAchievementsForPlayer()method 
public function getAchievementsForPlayer(playerID:String):void

Returns a list of Achievement objects to the IGetAchievementsCallback callback. The list returned in the response includes all visible achievements for the game. Visible achievements include those that were never hidden, and those that were once hidden but since made visible. Each Achievement object in the list includes the specified player's progress toward the Achievement.

Parameters

playerID:String — The playerId for the player we want to retrieve achievements data about.

updateProgress()method 
public function updateProgress(achievementID:String, achievementProgress:Number):void

Updates progress toward the specified Achievement by the specified amount. Result is returned to the IUpdateProgressCallback callback. If a value outside of range is submitted, it is capped at 100 or 0. If submitted value is less than the stored value, the update is ignored.

Parameters

achievementID:String — String that contains the identifier of the achievement.
 
achievementProgress:Number — A number between 0.0 and 100.0.
